# DetroitBecomeHumanResearch
A simple but multiple reseach about Detroit:Become Human,with varies of analytic methods
# 《底特律：成为人类》 数据分析项目

## 项目简介
本项目旨在深度剖析现象级电影互动游戏《Detroit: Become Human》，通过多维度的数据分析手段，探讨影视类游戏的现状与未来发展趋势。
核心围绕游戏的**历史价格、日活数据、用户评价、销量及用户画像**等维度展开可视化分析，并尝试通过机器学习预测评价的认同度。

## 文档说明
Detroit--所有使用的数据集

report.ipynb--数据处理与分析代码，图像统计分析

steam_review.py--爬虫代码，使用steam官方插件爬取评论数据

## 数据来源
| 分析维度 | 数据来源 | 说明 |
|---------|---------|------|
| 价格 / 日活 / 销量 | SteamDB | 历史价格、日活（DAU）、销量等核心运营数据 |
| 游戏类别 & 标签 | GitHub 开源数据集 | 游戏标签、大类归属 |
| 用户评论 | 自研爬虫 | 爬取 Steam 及社区评论、发布时间、地区等信息 |

## 分析内容与可视化

### 1. 游戏标签对应用户画像分析
- 游戏标签：**词云图**
- 游戏类型占比：在全品类游戏中的分布

### 2. 历史价格趋势
- 可视化：**折线图**
- 目标：展示历史价格变化（非日更）
- 数据：SteamDB

### 3. 日活数据可视化
- **Seaborn 热力图**
- 统计：每年日活总和对比

### 4. 价格 & 日活联动分析
- 可视化：**twinx 双轴图**
- 分析：
  - 降价对热度的**及时性 / 滞后性**
  - 日活提升幅度

### 5. 日评变化趋势
- 可视化：**折线图**
- 与日活、价格叠加分析
- 作为热度预测参考依据

### 6. 官方销量分析
- 销量漏斗图：800万 / 1000万 / 1100万 销量阶段

### 7. 玩家评价分析
- 评论地区分布：**饼图**（国内 / 国外）
- 相应游玩时长：**提琴图**

## 机器学习板块
- 目标：**预测评价有用数**
- 分析一条评论高认同度的前置条件
- 使用两种机器学习方法进行交叉比对，观察不同方法的分析效果

## 技术栈
- 数据采集：Python Requests、SteamDB
- 数据处理：Pandas、NumPy
- 可视化：Matplotlib、Seaborn、WordCloud
- 机器学习：Scikit-learn
