# DetroitBecomeHumanResearch
A simple but multiple reseach about Detroit:Become Human,with varies of analytic methods
# 《底特律：成为人类》 数据分析项目

## 项目简介
本项目旨在深度剖析现象级电影互动游戏《Detroit: Become Human》，通过多维度的数据分析手段，探讨影视类游戏的现状与未来发展趋势。
核心围绕游戏的**历史价格、日活数据、用户评价、销量及用户画像**等维度展开可视化分析，并尝试通过机器学习预测游戏日活趋势。

## 数据来源
| 分析维度 | 数据来源 | 说明 |
|---------|---------|------|
| 价格 / 日活 / 销量 | SteamDB | 历史价格、日活（DAU）、销量等核心运营数据 |
| 游戏类别 & 标签 | GitHub 开源数据集 | 游戏标签、大类归属 |
| 用户评论 | 自研爬虫 | 爬取 Steam 及社区评论、发布时间、地区等信息 |

## 分析内容与可视化

### 1. 历史价格趋势
- 可视化：**折线图**
- 目标：展示历史价格变化（非日更）
- 数据：SteamDB

### 2. 日活数据可视化
- 方案A：**3D 柱形图**
  - X：日
  - Y：月
  - Z：日活
  - 颜色：区分年份
- 方案B：**Seaborn 热力图**
- 方案C：**折线图**观察趋势
- 统计：每年日活总和对比

### 3. 价格 & 日活联动分析
- 可视化：**twinx 双轴图**
  - 左 Y：历史价格（柱形图）
  - 右 Y：日活（折线图）
- 分析：
  - 降价对热度的**及时性 / 滞后性**
  - 日活提升幅度

### 4. 日评变化趋势
- 可视化：**折线图 / 散点图**
- 可与日活、价格叠加分析
- 作为热度预测参考依据

### 5. 销量与用户画像
- 销量漏斗图：800万 / 1000万 / 1100万 销量阶段
- 评论地区分布：**饼图**（国内 / 国外）
- 游戏标签：**词云图**
- 游戏类型占比：在全品类游戏中的分布

## 机器学习板块
- 目标：**预测未来日活**
- 特征：
  - 价格、评论数、关注热度
  - 时代舆论、AI 话题热度
- 输出：日活预测曲线
- 使用两种机器学习方法进行交叉比对，观察不同方法的分析效果

## 技术栈
- 数据采集：Python Requests、SteamDB
- 数据处理：Pandas、NumPy
- 可视化：Matplotlib、Seaborn、WordCloud
- 机器学习：Scikit-learn
